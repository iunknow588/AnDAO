# AnDaoWallet H5 UI 设计参考文档

## 文档信息

- **项目名称**: AnDaoWallet HTML5 版本
- **文档版本**: 1.0
- **创建日期**: 2024年
- **文档状态**: 开发中

## 重要说明

⚠️ **本项目的 UI 设计仅参考 Keplr 钱包的设计风格和用户体验，代码完全独立实现，不依赖 Keplr 的任何代码。**

## 设计原则

1. **参考 Keplr 的视觉风格**：颜色、字体、间距、布局
2. **参考 Keplr 的交互模式**：用户操作流程、反馈机制
3. **代码完全独立**：不依赖 Keplr 的任何代码或组件
4. **基于 React + Styled Components**：使用现代前端技术栈独立实现

## Keplr 钱包 UI 分析

### 1. 整体布局

#### 1.1 主界面结构
- **顶部导航栏**：包含钱包名称、账户信息、设置入口
- **侧边栏**（可选）：功能导航菜单
- **主内容区**：当前页面内容
- **底部状态栏**（可选）：网络状态、连接状态

#### 1.2 颜色方案
- **主色调**：深蓝色系（#4c6ef5 类似）
- **背景色**：白色/浅灰色
- **文字颜色**：深灰色（#333）
- **强调色**：蓝色（用于按钮、链接）
- **警告色**：橙色/红色（用于警告信息）

#### 1.3 字体
- **主字体**：系统默认字体栈（San Francisco、Roboto 等）
- **标题字体**：加粗、较大字号
- **正文字体**：常规、中等字号
- **辅助文字**：较小字号、浅色

### 2. 页面设计

#### 2.1 首页/资产页面
- **账户地址显示**：可复制、可查看二维码
- **余额显示**：大号数字、清晰可见
- **代币列表**：卡片式布局、支持滚动
- **操作按钮**：发送、接收、设置等

#### 2.2 创建账户页面
- **步骤指示器**：显示当前步骤
- **输入表单**：清晰的标签和提示
- **按钮设计**：主要操作按钮突出显示
- **进度反馈**：加载状态、成功/失败提示

#### 2.3 发送交易页面
- **收款地址输入**：支持粘贴、扫描二维码
- **金额输入**：数字键盘、余额显示
- **Gas 设置**：可选、默认推荐值
- **确认按钮**：二次确认机制

#### 2.4 交易历史页面
- **交易列表**：时间排序、状态标识
- **筛选功能**：按状态、类型筛选
- **详情查看**：点击查看交易详情
- **外部链接**：链接到区块浏览器

#### 2.5 设置页面
- **分组显示**：功能分组、清晰分类
- **开关控件**：布尔设置使用开关
- **选择控件**：多选项使用下拉或单选
- **操作按钮**：危险操作需要确认

### 3. 组件设计

#### 3.1 按钮组件
- **主要按钮**：蓝色背景、白色文字、圆角
- **次要按钮**：白色背景、蓝色边框、蓝色文字
- **危险按钮**：红色背景或红色文字
- **禁用状态**：灰色、不可点击

#### 3.2 输入框组件
- **边框样式**：细边框、圆角
- **焦点状态**：蓝色边框高亮
- **错误状态**：红色边框、错误提示
- **占位符**：浅灰色文字

#### 3.3 卡片组件
- **背景色**：白色
- **阴影**：轻微阴影、增加层次感
- **圆角**：统一圆角值
- **内边距**：合适的间距

#### 3.4 模态框组件
- **遮罩层**：半透明黑色背景
- **内容区**：居中显示、白色背景、圆角
- **关闭按钮**：右上角或底部
- **动画效果**：淡入淡出

### 4. 交互设计

#### 4.1 反馈机制
- **加载状态**：加载动画、进度指示
- **成功提示**：绿色提示、短暂显示
- **错误提示**：红色提示、详细错误信息
- **确认对话框**：重要操作需要确认

#### 4.2 动画效果
- **页面切换**：平滑过渡
- **按钮点击**：轻微缩放反馈
- **列表滚动**：流畅滚动
- **加载动画**：旋转或进度条

#### 4.3 响应式设计
- **移动端适配**：触摸友好的按钮大小
- **平板适配**：充分利用屏幕空间
- **桌面端适配**：合理的最大宽度
